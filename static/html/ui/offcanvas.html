  1. Off-canvas

Off-canvas

Off-canvas layers slide in from either the left or the right. They span the full height of the screen. You can use multiple off-canvas displays per page.

Properties

Positioning

You can place an off-canvas either on the left or the right side off the screen.

Open left off-canvas Open right off-canvas

Size

An off-canvas can have any size. Material Admin has a number of standard sizes you can choose, but you can also set the width in a style attribute for easy customization.

Size 1 Size 2 Size 3 Size 4
Some sizes as an example

Colors

Every color that is available for a card, is also available for an off-canvas.

Color 1 Color 2 Color 3 Color 4
Basic headings

Backdrop

A backdrop is a layer on top of the content. This layer is clickable so that the off-canvas is closed. You can disable the backdrop by adding data-backdrop="false".
